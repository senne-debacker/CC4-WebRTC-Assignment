# CC4-WebRTC-Assignment

## Concept: Simon Says

**Description**
A digital version of the famous game 'Simon Says'. The user connects to the laptop by scanning a QR-code and starts the game. The laptop will show commands on the screen. If the command is preceded by the text 'Simon Says', the user has to execute the command, of not, they cannot execute the command. If they fail before the timer ends, it is game over. 

**Commands**

Using screen
- Tap the screen
- Draw something on the screen
- ...

Using gyroscope/accelerometer
- Shake
- Jump
- Duck
- Do a 360
- Pour me a drink
- ...

Using microphone
- Yell/talk
- ...

Using camera
- ...
