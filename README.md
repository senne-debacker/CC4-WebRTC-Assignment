# CC4-WebRTC-Assignment

## Concept: Simon Says

**Description**
A digital version of the famous game 'Simon Says'. The user connects to the laptop by scanning a QR-code and starts the game. The laptop will show commands on the screen. If the command is preceded by the text 'Simon Says', the user has to execute the command, of not, they cannot execute the command. If they fail before the timer ends, it is game over. 

**Commands**

Using screen
- Tap the screen
- Draw something on the screen
- (AI) Smile
- ...

Using gyroscope/accelerometer
- Shake
- Jump
- Duck
- Do a 360
- Pour me a drink
- ...

Using microphone
- Yell/talk
- ...

Using camera
- ...

const clients = {};

io.on('connection', socket => {
  clients[socket.id] = { id: socket.id };
  io.emit('clients', clients);
  io.emit('client-connection', clients[socket.id]);

  socket.on('disconnect', () => {
    io.emit('client-disconnect', clients[socket.id]);
    delete clients[socket.id];
    io.emit('clients', clients);
  });

  // Forward WebRTC signaling between peers
  socket.on('signal', (peerId, signal) => {
    io.to(peerId).emit('signal', peerId, signal, socket.id);
  });
});

// sender.html — devicemotion registers acceleration and feeds shake/jump/duck detectors
let lastAccel = { ax: 0, ay: 0, az: 0 };

window.addEventListener("devicemotion", (event) => {
  if (!sensorsActive) return;

  // Capture raw acceleration for telemetry
  const a = event.acceleration || event.accelerationIncludingGravity;
  if (a) {
    lastAccel = {
      ax: a.x?.toFixed(2) ?? 0,
      ay: a.y?.toFixed(2) ?? 0,
      az: a.z?.toFixed(2) ?? 0,
    };
  }

  // Shake: require 4+ direction reversals in X or Y within 1.2 s
  const linAccel = event.acceleration;
  if (linAccel) {
    const val = Math.abs(linAccel.x || 0) > Math.abs(linAccel.y || 0)
      ? (linAccel.x || 0) : (linAccel.y || 0);
    const sign = val > SHAKE_THRESH ? 1 : val < -SHAKE_THRESH ? -1 : 0;
    if (sign !== 0 && sign !== shakeLastSign) {
      shakeReversals++;
      shakeLastSign = sign;
      if (shakeReversals >= SHAKE_REVERSALS_NEEDED) sendGesture("shake");
    }
  }
});
